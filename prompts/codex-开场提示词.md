# Codex 开场提示词

请先阅读仓库根目录下的 `AGENTS.md`、`README.md` 和 `docs/` 目录中的文档，再开始工作。

当前项目是课程实验项目，目标是实现一个 Web 版 Scrum 过程管理平台，只优先完成基本版功能，不做课程要求之外的复杂扩展。

请你先完成以下事情，不要直接大改代码：

1. 扫描当前仓库结构
2. 总结已有模块、技术栈、启动方式和未完成部分
3. 对照 `docs/API接口草案.md` 与 `docs/数据库设计.md`，检查当前实现与设计是否一致
4. 给出一个按优先级排序的开发建议清单
5. 标出最值得先做的 3 个任务

输出格式要求：

- 当前项目结构摘要
- 已完成内容
- 缺失内容
- 风险点
- 下一步建议

如果你发现仓库里没有代码，说明最小可落地的初始化方案，并列出建议创建的目录和文件。
