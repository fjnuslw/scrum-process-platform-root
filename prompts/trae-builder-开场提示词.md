# Trae Builder 开场提示词

这是一个课程实验项目，请帮我开发一个 Web 版 Scrum 过程管理平台。

你在开始生成代码前，请先阅读根目录中的以下文档：

- `README.md`
- `docs/项目执行与技术说明.md`
- `docs/数据库设计.md`
- `docs/API接口草案.md`
- `docs/开发任务清单.md`

开发约束如下：

1. 只做基本版功能，不做增强版
2. 技术栈优先使用 Vue 3 + Vite + Element Plus + Node.js + Express + MySQL
3. 保持结构简单，优先保证课程作业可运行、可测试、可解释
4. 不要引入权限系统、复杂鉴权、消息通知、微服务、复杂状态管理
5. 每次只完成一个模块，并说明修改了哪些文件、如何运行和如何验证

请先输出：

1. 项目初始化方案
2. 建议的前后端目录结构
3. 第一阶段要生成的文件清单
4. 预计先完成的第一个模块

在我确认前，不要一次性生成全部项目代码。
